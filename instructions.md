*make sure that you have an Open AI and Google Maps API Key

1) poetry install

2)  Create a .env file in the project root so it can have the following:
OPENAI_API_KEY=YOUR_OPENAI_KEY
GOOGLE_MAPS_API_KEY=YOUR_GOOGLE_KEY
OPENAI_MODEL=gpt-4o-mini

3) poetry run python main.py

4) Follow the prompts in the terminal. To exit at any time, type: exit or quit
